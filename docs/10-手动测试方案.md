# 手动测试方案 — 学习助手 v2.4

> 本文档供你直接执行。每个测试用例包含操作步骤和预期结果，`[实际结果]` 处留空，你测试时自行记录。

---

## 前置准备

- [ ] 确认 Python 3.9+ 已安装
- [ ] 执行 `pip install -r requirements.txt`（首次需要）
- [ ] 检查 `.env` 已填入 LLM_API_KEY
- [ ] 执行 `python backend/app.py`，看到日志输出 `学习助手 v2.4 启动中...`
- [ ] 浏览器打开 http://127.0.0.1:5000
- [ ] 准备一份测试文件（`test_learning.md`，内容为 200-500 字的任何学习资料，如 AI/产品/编程知识）
