# Changelog

本文件记录项目的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/)，版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [2.5] — 2026-07-16

### Added
- 统一向量集合：每用户一个 ChromaDB collection，跨文档检索一次 query
- 语义切分升级：中文句感知 + 表格保护（chunk=800, overlap=150, 6 级分隔符递归）
- 长期用户记忆系统：对话后 LLM 提取学习画像 → system prompt 注入 → "AI 眼中的你" 卡片
- 掌握度阈值优化：L2 备选路径 + L3 薄弱点消除判定
- ONNX 可选加速支持
- 文档去重合并 + README 升级 + 版本号统一

### Changed
- 向量集合从按文档拆分改为按用户统一
